Ordered List of HTML Tags

  1. <html> - This tag represents the root element of an HTML document. It contains all other HTML elements (except for the DOCTYPE declaration).
  2. <head> - The <head> tag contains meta-information about the document, such as its title, character encoding, linked stylesheets, and scripts.
  3. <title> - This tag defines the title of the document, which appears in the browser's title bar or tab.
  4. <body> - The <body> tag contains the content of the document, such as text, images, and other media, which is visible to the user.
  5. <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - These are header tags used to define headings. <h1> is the highest level, and <h6> is the lowest.
  6. <p> - The <p> tag defines a paragraph of text.
  7. <a> - The <a> tag is used to create hyperlinks. It can be used to link to other web pages or resources.
  8. <img> - The <img> tag is used to embed images in a webpage. It is a self-closing tag and requires the src attribute to specify the image source.
  9. <ul> - The <ul> tag defines an unordered list. List items inside it are defined with the <li> tag.
  10. <ol> - The <ol> tag defines an ordered (numbered) list. It also uses <li> elements to define each list item.
  11. <li> - The <li> tag represents a list item. It is used within both <ul> (unordered lists) and <ol> (ordered lists).
  12. <div> - The <div> tag is a block-level container used to group elements and structure a webpage.
  13. <span> - The <span> tag is an inline container used to group text or other inline elements, often used with CSS for styling.
  14. <form> - The <form> tag defines an HTML form for user input. It can contain various input elements like text fields, buttons, and checkboxes.
  15. <input> - The <input> tag is used to create interactive controls in a form, like text boxes, checkboxes, and radio buttons.
  16. <button> - The <button> tag defines a clickable button that can submit a form or trigger JavaScript actions.
  17. <table> - The <table> tag is used to create a table. It contains rows and columns of data.
  18. <tr> - The <tr> tag defines a row in a table.
  19. <td> - The <td> tag defines a cell in a table row.
  20. <th> - The <th> tag defines a header cell in a table, typically rendered in bold and centered.
  21. <iframe> - The <iframe> tag is used to embed an external webpage within the current page. It creates an inline frame.
  22. <link> - The <link> tag is used to link external resources, such as stylesheets, to the HTML document. It is commonly used in the <head> section.
  23. <meta> - The <meta> tag provides metadata about the document, such as the character set or author. It is used in the <head> section.
  24. <script> - The <script> tag is used to define JavaScript code within an HTML document. It can be placed either in the <head> or <body> sections.
